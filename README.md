# Connect-4

A simple, classic Connect-4 game built using JavaScript, HTML, and CSS.


## Features

- Play Connect-4 with two players (Red vs Yellow)
- Interactive web interface
- Visual board updates after each move
- Win detection for vertical, horizontal, and diagonal connections

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Natt1111/Connect-4.git
   ```
2. **Open the game**
   - Open `index.html` directly in your web browser.
   - No installation or dependencies required!

## Technologies Used

- JavaScript (86%)
- CSS (7%)
- HTML (6%)

## How to Play

- Take turns dropping colored discs into a grid.
- The first player to connect four of their pieces in a line (vertically, horizontally, or diagonally) wins.
